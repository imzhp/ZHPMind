# Listing 优化（Listing Optimization）

Listing 是亚马逊商品详情页的核心载体，是[[a9-algorithm|A9 算法]]转化率与相关性两大支柱的直接体现。优化 Listing 的本质是：**在算法规则内，最大化匹配买家搜索意图，并将点击转化为订单**。

## 打造前的准备工作

### 第一步：理解排序逻辑（Sort By 分析）

在亚马逊前台搜索目标关键词，依次查看四种排序：

| 排序方式 | 分析目的 |
|----------|---------|
| Featured（相关性）| 了解默认排名逻辑，即 A9 综合权重 |
| Price（价格）| 了解价格区间分布，规划自己的价格定位 |
| Avg. Customer Review（平均评论）| 了解打该词所需的最低评论数量基准 |
| Newest Arrivals（最新上架）| 快速找到新上架竞品；新品期从这里进入是有加权的 |

### 第二步：选定参考对象（竞品拆解）

**合适竞争对手的条件**：
- 选取 3 个 Best Seller 作为核心参考
- 评论数量、价格区间与自身接近
- 在目标关键词下有稳定排名

**拆解维度**：
1. 标题结构与关键词布局
2. 图片数量与内容（主图、场景图、卖点图、材质图）
3. 五点描述的卖点框架
4. A+ 内容结构
5. 参数完整性（所有属性字段是否填满）
6. 评论中的高频词（买家核心诉求）

**拆解汇总**：将各竞品优点整合为"10X 内容库"，作为自己 Listing 的内容基础。

### 第三步：AIDA 模型诊断

用 AIDA 框架诊断现有 Listing 的短板：

| 阶段 | 对应 Listing 元素 | 问题诊断 |
|------|------------------|---------| 
| Awareness（认知）| 主图、价格、标题前 70 字符 | 能否在搜索结果页被注意到？ |
| Interest（兴趣）| 次图、A+ 首屏 | 点进来后能否激发兴趣继续看？ |
| Desire（欲望）| 五点、A+ 详细内容、QA、视频 | 能否产生购买欲望？ |
| Action（行动）| 价格、Coupon、促销标签、评论星级 | 能否最终下单？ |

## 标题（Title）

**核心原则**：
- 前 70 个字符：手机端显示范围，必须包含最核心的品牌词 + 关键词 + 卖点
- 品牌词位置：写在最前面可增加品牌相关性；若品牌不知名，可将核心词放首位
- 严格遵守字数上限，超字数可能被降权
- 参考 BA（Brand Analytics）数据中的高流量词，优先埋入标题
- 能连续出现的关键词尽量连续（分散则需在五点 / QA 中再次埋词加权）

**进阶技巧**：在核心关键词前添加需求形容词（如 `Strong Magnetic`、`Breathable`），创造差异化点击率，且不影响主推词权重。

## 五点描述（Bullet Points）

- 结构化呈现核心卖点，每点聚焦一个主题
- 埋入标题中未出现的重要关键词
- 参考竞品高频 Review 词语，击中买家核心痛点

## 图片（Images）

**主图（Main Image）**：
- 影响约 80% 的点击率（CTR）
- 白底、清晰、主体突出；可做合规范围内的小动作

**次图策略**：
- 第 2–3 张：场景图（使用场景 + 目标用户）
- 卖点对比图、材质图（专门用一张图解释材质，效果显著）
- 图片数量上满（通常 7–9 张）
- 不同变体可分别上传视频，增加关联曝光

**图片与评论的关系**：
- 视频 > 图片 > 文字评论（权重与保护星级效果）
- 新品期首先安排带图片和视频的评论，保护星级不被单条差评拉低

## 搜索词（Search Terms / ST）

- 后台隐藏字段，补充前台无法自然放入的关键词
- 泛流量词（如 `car accessories`、`gift for mom`）优先放入 ST，通过自动广告以低 Bid 跑出来
- 新品早期资源有限时，ST 可暂不填写，专注核心词；成熟后再补泛词

## QA 版块

- 主动设置 QA，增加核心关键词密度
- 把 14 个编辑推荐词（Editorial Recommendation 词）埋入 QA 标题
- QA 标题可复用到 Post 标题、文案场景

## A+ 内容

- 有品牌注册后可用
- 图片 Alt 标签（Alt Tag）可埋关键词（通过浏览器检查 → 搜索 Alt 验证）
- 前期泛流量词可先放入 A+，待 Listing 成熟后再移至 ST

## 评论（Review）管理

**权重层级**（由高到低）：
> 视频 > 图片 > 文字 ｜ 最新评论 > 老评论 ｜ 差评 > 好评 ｜ VP（Verified Purchase）> 直评

**评论卡位策略**：
- 首页展示评论数量取决于评论长度
- 用带视频 + 10 张图片 + 2500 字的高质量评论，可将首页锁定为 2 条
- 新品期尽早安排 3 星以上评论，防止 1 星评论杀伤力过大

**AC 标（Amazon's Choice）获取**：
- 仅与转化率相关
- 通过关键词购买链接（带关键词的 URL）送测，前期做 3–4 个 100% 转化
- 效益：额外标签 + 额外流量入口 + 提升点击率

## 关键词收集体系

| 工具 / 来源 | 特点 | 优先级 |
|------------|------|--------|
| BA（Brand Analytics）| 真实 180 万搜索数据，最权威 | ⭐⭐⭐⭐⭐ |
| 编辑推荐词（Editorial Recommendation）| 高质量流量词，可联系编辑争取推荐位 | ⭐⭐⭐⭐⭐ |
| H10 反查 AC 词 | 竞争弱、转化率高 | ⭐⭐⭐⭐ |
| SIF 拓展词 | 竞争对手 ASIN 的关键词影响力图谱 | ⭐⭐⭐⭐ |
| Related Searches（相关搜索）| 亚马逊前台推荐，优质词来源 | ⭐⭐⭐ |
| 第三方工具 | 数据库词，不一定准确 | ⭐⭐ |

**埋词优先顺序**：
1. 标题（最高权重，尽量连续出现）
2. 五点描述
3. QA
4. ST（泛词、长尾词）
5. A+（泛词、前期测试）

## 价格策略

- **Business Price（企业购价格）**：比普通价低 1% 或 1 分钱，才会在 B2B 流量入口显示
- **价格区间布局**：用 1-pack / 2-pack / 10-pack 等变体，占据不同价格区间的流量入口
- 定价决定目标客群：高价筛选高质量客户，低价吸引价格敏感型客户

## 品牌旗舰店（Brand Store）

- Prime Day 期间装修旗舰店的品牌销量增长高达 **646%**
- 旗舰店转化率比单独 Listing 高 **2 倍以上**（尤其新品）
- 可作为站外流量**着陆页（Landing Page）**，避免直接导 Listing 拉低整体转化率
- 通过 Post 免费引流（文案埋词，每日发布）
- 积累 Followers → 开通邮件营销（Customer Engagement，内测功能）
- SB 广告效果最佳落地页，配合品牌故事

## Listing 销量下滑的原因排查

1. 账号受到处罚（可能是暗中降权）
2. 产品生命周期已进入衰退期，市场饱和
3. Listing 长期未优化
4. 短期断货、差评、退款、A-to-Z 投诉激增
5. 竞争对手销量增加，占据靠前排名
6. 竞争对手短期内多次参加秒杀活动

> 防护建议：设置最大购买量限制，防止竞争对手恶意买空库存制造差评。
